**NDEMBO KIN — SPORT**

Design System — Football & Leadership

*Tournois • Compétitions • Événements Panafricains*

*Version 1.0 Sport | Juin 2026*

# **1\. Vision Sport & Leadership**

ndembo Kin mobilise le football et le sport comme leviers de transformation du leadership en Afrique. Notre identité visuelle incarne l'énergie, la compétition, et l'unité continentale.

*Le sport ne fait pas que divertir — il inspire, unit et élève les leaders de demain.*

## **Piliers de l'identité Sport**

* **Excellence & Performance** — Slate Blue (professionnalisme)

* **Action & Dynamisme** — Red (énergie, passion)

* **Victoire & Succès** — Gold (triomphe)

* **Unité Panafricaine** — Cyan (connexion continentale)

# **2\. Palette de Couleurs Sport**

Optimisée pour les événements sportifs, les compétitions et les tournois. Les couleurs évoquent l'énergie, l'excellence et le prestige.

**Marine (Slate Blue) \+ Red (action) \+ Gold (victoire) \= Identité Sport**

| Couleur | Nom | Code HEX | Utilisation Sport |
| :---- | :---- | :---- | :---- |
|  | **Slate Blue** | 3D5A6C | Uniformes, scores, texte |
|  | **Red** | E91E1E | Logo action, énergie, CTA |
|  | **Gold** | F4A800 | Victoires, trophées, podiums |
|  | **Cyan** | 1E9FD8 | Accents, équipes, dynamisme |
|  | **White** | FFFFFF | Fond, jersey, clarté |
|  | **Dark Gray** | 2C3E50 | Texte dense, tableaux scores |

## **Combinaisons de Couleurs Recommandées**

* **Tournoi Principal** : Slate Blue \+ Red (énergie compétitive)

* **Événement d'Excellence** : Slate Blue \+ Gold (prestige)

* **Compétition Continentale** : Red \+ Cyan (unité africaine)

* **Victoire/Podium** : Gold sur Slate Blue (cérémonies)

# **3\. Typographie Sport**

Forte, impactante, lisible. Conçue pour les affiches, les retransmissions en direct et les événements de grande envergure.

## **Familles Typographiques**

**Display Sport**

* **Oswald Bold, Roboto Black ou Impact**

* Usage : Titres tournoi, noms équipes, scores majeurs, affiches

**Corps Lisible**

* **Open Sans, Roboto ou Lato**

* Usage : Description matchs, règlements, communiqués

**Données/Scores**

* **JetBrains Mono ou Courier New**

* Usage : Scores directs, statistiques, classements

## **Hiérarchie Typographique Sport**

| Usage | Taille | Poids | Couleur | Exemple |
| :---- | :---- | :---- | :---- | :---- |
| Titre Tournoi | 48–72pt | Black | Red | « TOURNOI 2026 » |
| Score | 40–60pt | Bold | Slate Blue | « 3-2 » |
| Équipe | 28–36pt | Bold | Red | « LIONS DAKAR » |
| Détails | 14–16pt | Regular | Gris | Min. 45 | Terrain |

# **4\. Logo Sport & Variantes**

Le logo ndembo Kin incarne trois dimensions du leadership sportif :

* **Robot/Technologie** \= Analyse, tactique, innovation

* **Cœur multicolore** \= Passion, fair-play, humanité

* **Joueur \+ Afrique** \= Excellence continentale, action

## **Utilisation dans le Contexte Sport**

**Logo Principal (version couleur)**

* Affiches de tournoi, billetterie, sites de compétition

**Version Monochromatique (Red)**

* Uniformes d'équipe, t-shirts, merchandise

**Version White (sur Slate Blue)**

* Stades, tableaux de bord, retransmissions

**Favicon (carré compact)**

* Applications mobile, onglets navigateur

## **Règles Non-Negotiables**

* Espacement minimal \= 1/4 de hauteur du logo

* Taille minimale : 100px web, 1.5cm print

* Jamais de rotation, distorsion ou effet 3D

* Fond blanc/transparent favori ; Slate Blue accepté

# **5\. Composants & Éléments Sport**

## **Cartes de Match**

Design : En-tête Red, corps blanc, scores Slate Blue bold, texte détails gris moyen. Icône maillot pour chaque équipe.

## **Badges & Médailles**

* **Or** \= Victoire (1ère place)

* **Argent** \= Vice-champion

* **Bronze** \= Troisième place

## **Tableaux de Classement**

En-tête Slate Blue, texte blanc. Alternance lignes blanc/light gray. Colonnes : Rang (Red), Équipe, Points, Différence. Icônes trophée pour top 3\.

## **Boutons d'Action**

* **Primaire (Red)** — S'inscrire, Acheter Billet

* **Secondaire (Cyan)** — En savoir plus, Voir détails

* **Danger/Désinscription (Red clair)** — Retrait, Annulation

# **8\. Grille & Espacement**

Système 8px pour web, 2.8mm pour print. Crée harmonie et cohérence visuelle sur tous les médias.

| Échelle | Valeur | Utilisation Sport |
| :---- | :---- | :---- |
| xs | 4px | Icônes maillots, séparateurs |
| sm | 8px | Padding boutons, équipes |
| md | 16px | Entre matchs, équipes |
| lg | 24px | Entre phases/jours |
| xl | 40px | Hero tournoi, sections majeures |
